## 1. 交互哲学：Fluid & Intuitive（流畅直觉）

### 1.1 核心原则

**让交互如水般自然流动，让意图在指尖清晰表达**

| 原则 | 具体表现 | 避免 |
|------|----------|------|
| **滑动优先** | 核心浏览体验使用滑动手势 | 大量点击跳转 |
| **手势连贯** | 手势动作与视觉反馈同步 | 延迟或断层 |
| **上下文敏感** | 同一元素在不同场景有不同交互 | 死板的单一交互 |
| **触感呼应** | 关键操作有触觉反馈 | 安静的交互 |
| **物理隐喻** | 遵循现实世界物理规律 | 突兀的数字化动作 |

### 1.2 交互分类体系

```
┌─────────────────────────────────────────────────┐
│  核心交互层级                                    │
├─────────────────────────────────────────────────┤
│  L1 - 导航流动：上下滑动、Tab切换               │
│  L2 - 内容操作：点击展开、左右滑动快捷操作       │
│  L3 - 深度交互：长按、拖拽、捏合                │
│  L4 - 微交互：悬停、弹性反馈、粒子效果           │
└─────────────────────────────────────────────────┘
```

## 2. 全局导航交互

### 2.1 Tab 切换（底部悬浮玻璃栏）

**交互方式：点击 / 左右滑动切换**

#### 点击交互

| 操作 | 触发条件 | 视觉反馈 | 动画 | 触觉 |
|------|----------|----------|------|------|
| 点击 Tab 图标 | 单击未选中 Tab | 图标放大 1.1x | `matchedGeometryEffect` 指示器移动（0.3s） | `.impact(.light)` |
| 重复点击当前 Tab | 单击已选中 Tab | 页面回到顶部 | 列表滚动动画（0.5s） | `.impact(.medium)` |

**动画细节：**

```swift
// Tab 切换动画
.transition(.asymmetric(
    insertion: .move(edge: .trailing).combined(with: .opacity),
    removal: .move(edge: .leading).combined(with: .opacity)
))
.animation(.spring(response: 0.4, dampingFraction: 0.8), value: selectedTab)

// 指示器移动
Circle()
    .frame(width: 6, height: 6)
    .matchedGeometryEffect(id: "tabIndicator", in: namespace)
    .animation(.spring(response: 0.3, dampingFraction: 0.7), value: selectedTab)
```

#### 左右滑动切换（全局手势）

| 手势 | 效果 | 实现 |
|------|------|------|
| 向左滑动 | 切换到下一个 Tab | `.gesture(DragGesture().onEnded { ... })` |
| 向右滑动 | 切换到上一个 Tab | 判断 `translation.width` |

**滑动条件：**
- 滑动距离 > 屏幕宽 25%
- 或滑动速度 > 300pt/s

**视觉反馈：**
```
滑动中 → 两个Tab页面同时显示（上一页向左移动，新页面从右侧进入）
完成 → 弹性回弹到位，指示器同步移动
取消 → 原页面弹性归位
```

### 2.2 平台彩带选择器（PlatformRibbon）

**交互方式：点击 / 水平滑动浏览**

#### 点击选择

| 操作 | 视觉反馈 | 动画时长 | 触觉 |
|------|----------|----------|------|
| 点击平台名 | 文字发光 + 渐变下划线移动 | 0.3s | `.impact(.light)` |
| 选中新平台 | 背景氛围色过渡 | 0.5s | 无 |

**下划线动画：**

```swift
// 渐变下划线
Capsule()
    .fill(platformGradient)
    .frame(height: 3)
    .matchedGeometryEffect(id: "platformUnderline", in: namespace)
    .shadow(color: platformColor.opacity(0.6), radius: 4)
```

#### 水平滑动浏览

- **惯性滚动**：ScrollView 自然减速
- **边界弹性**：ScrollView bounces
- **指示器**：无（列表两端自然淡出）

## 3. 核心卡片交互（MorphicCard）

### 3.1 卡片状态机

```
┌──────────────┐   点击   ┌──────────────┐   松手   ┌──────────────┐
│   默认态     │ ────────>│   按下态     │ ────────>│   详情页     │
│  Light阴影   │          │  缩放0.98    │          │  展开动画    │
└──────────────┘          └──────────────┘          └──────────────┘
       ↑                                                    │
       │                                                    │
       └────────────────── 返回手势 ────────────────────────┘
```

### 3.2 点击进入详情

**手势：单击卡片主体区域**

| 阶段 | 视觉变化 | 动画参数 |
|------|----------|----------|
| 按下 | 缩放 0.98，阴影加深，3D 倾斜 | `.spring(response: 0.2)` |
| 松手 | 卡片"生长"成全屏 | `matchedGeometryEffect` |
| 展开 | 背景渐变到平台色，内容淡入 | 0.4s easeInOut |
| 曲线 | 迷你曲线"展开"成完整曲线 | 0.6s spring |

**实现关键：**

```swift
// 卡片生长动画
cardView
    .matchedGeometryEffect(id: topic.id, in: namespace)
    .zIndex(1)

// 背景过渡
backgroundGradient
    .opacity(isDetail ? 1.0 : 0.05)
    .animation(.easeInOut(duration: 0.4), value: isDetail)
```

### 3.3 左滑快捷操作（卡片级）

**手势：在卡片上向左滑动**

| 滑动距离 | 显示操作 | 继续滑动 | 松手效果 |
|----------|----------|----------|----------|
| 0-40pt | 收藏按钮显露 | 分享按钮显露 | < 50% 回弹，> 50% 展开 |
| 40pt+ | 收藏按钮完全显示 | 收藏 + 分享 | 停留在展开态 |

**操作按钮：**

```
┌───────────────────┬───┬───┐
│  卡片内容 ←滑动   │❤️ │↗️ │
│                   │收藏│分享│
└───────────────────┴───┴───┘
```

**视觉反馈：**
- 按钮背景：平台渐变色 + 玻璃材质
- 图标：白色，触摸时放大 1.2x
- 滑动中：卡片轻微倾斜（3D 效果）

### 3.4 右滑快捷操作（可选）

**手势：向右滑动**

| 滑动距离 | 效果 | 用途 |
|----------|------|------|
| 80pt+ | 屏蔽此话题 | 显示"屏蔽"按钮 |

**确认流程：**
1. 滑动超过阈值 → 按钮变红，震动反馈 `.impact(.medium)`
2. 松手 → Toast 提示"已屏蔽，可在设置中恢复"
3. 卡片淡出 + 向右飞出屏幕（0.3s）

### 3.5 长按操作

**手势：长按卡片主体 0.6s**

**触发效果：**

```
弹出圆形 Context Menu（玻璃材质）
┌─────────────────┐
│  📌 置顶         │
│  ❤️  收藏        │
│  🚫 屏蔽        │
│  ↗️  分享        │
└─────────────────┘
```

**菜单动画：**
- 进入：缩放从 0.8 → 1.0 + 弹性
- 背景：模糊背景遮罩（60% 透明度）
- 触觉：`.impact(.heavy)`

## 4. 列表滚动交互

### 4.1 垂直滚动（核心浏览）

**手势：上下滑动列表**

| 滚动速度 | 视觉效果 | 性能优化 |
|----------|----------|----------|
| 慢速 | 视差效果：背景移动速度 0.3x | LazyVStack |
| 快速 | 卡片轻微模糊（速度感） | `.drawingGroup()` |

**滚动指示器：**
- 无传统 scrollbar
- 快速滚动时右侧出现"渐变进度带"（平台色）

### 4.2 下拉刷新

**手势：顶部下拉**

| 阶段 | 拉动距离 | 视觉反馈 | 动画 |
|------|----------|----------|------|
| 准备 | 0-60pt | 顶部平台色光晕逐渐变亮 | 光晕半径随距离增大 |
| 触发 | 60pt+ | 光晕脉冲一次 | `.impact(.medium)` |
| 刷新中 | — | 旋转渐变环（平台色） | 持续旋转 |
| 完成 | — | 光晕爆散成粒子 | 粒子向下飘散 0.8s |

**技术实现：**

```swift
.refreshable {
    await viewModel.refresh()
}
.refreshableStyle(.platformPulse) // 自定义样式
```

### 4.3 触底加载更多

**触发：滚动到底部倒数第 3 个项目**

**视觉反馈：**
- 底部出现"加载指示器"（平台色小点脉冲）
- 新内容淡入 + 轻微上浮动画

## 5. 详情页交互

### 5.1 页面进入/退出

**进入：** 点击卡片（见 3.2）

**退出方式 1：向下滑动返回**

| 手势 | 效果 | 动画 |
|------|------|------|
| 从顶部向下滑 | 页面跟随手指下移 | 实时跟手 |
| 滑动距离 > 20% | 松手后自动关闭 | `matchedGeometryEffect` 回缩 |
| 滑动距离 < 20% | 弹性回弹 | `.spring(response: 0.3)` |

**退出方式 2：点击关闭按钮**

- 左上角"✕"按钮
- 点击 → 缩放 0.95 → 页面回缩到卡片位置

### 5.2 热度曲线交互（完整版）

**手势 1：水平拖动光标**

| 操作 | 视觉反馈 | 数据显示 |
|------|----------|----------|
| 手指按住曲线区域 | 垂直虚线光标出现 | 气泡显示数据点 |
| 水平拖动 | 光标跟随移动 | 气泡数据更新 |
| 松手 | 光标和气泡淡出 | 0.2s 延迟淡出 |

**气泡样式：**

```
     ╭─────────────╮
     │  1.2M 热度  │
     │  #3 排名    │
     │ 14:30 时间  │
     ╰──────┬──────╯
            │
         ───┼───  ← 垂直虚线
```

**手势 2：捏合缩放时间范围**

| 手势 | 效果 | 范围切换 |
|------|------|----------|
| 双指捏合 | 时间轴缩小（看更长历史） | 24h → 7d → 30d |
| 双指展开 | 时间轴放大（看更细节） | 30d → 7d → 24h |

**切换动画：**
- 曲线平滑变形（0.4s spring）
- X 轴刻度淡出/淡入

### 5.3 分享/收藏操作

**分享按钮：**

| 操作 | 反馈 | Sheet 内容 |
|------|------|------------|
| 点击分享图标 | 按钮脉冲放大 | 系统分享面板 |

**收藏按钮：**

| 操作 | 反馈 | 动画 |
|------|------|------|
| 点击空心❤️ | 填充变红 + 放大 1.3x | 粒子从心形四散 |
| 再次点击 | 空心 + 缩小 | 无粒子 |

**粒子效果：**
- 8-12 个小心形
- 颜色：平台渐变色
- 轨迹：向四周扩散，带旋转
- 时长：0.8s，逐渐透明消失

## 9. 特殊手势汇总

### 9.1 全局手势

| 手势 | 作用域 | 效果 | 优先级 |
|------|--------|------|--------|
| 左右滑动 | 主 Tab 内容区 | Tab 切换 | 低（被卡片滑动覆盖） |
| 三指左滑 | 全局 | 撤销操作（可选） | 最高 |
| 双击状态栏 | 全局 | 回到顶部 | 系统默认 |

### 9.2 手势优先级规则

```
卡片左滑快捷操作 > 详情页手势 > Tab 切换手势
    ↓                   ↓                ↓
    高优先级          中优先级          低优先级
```

**冲突解决：**
- 卡片区域：优先检测左滑 → 未触发则传递给 Tab 切换
- 使用 `.simultaneousGesture()` 和 `.highPriorityGesture()`

## 10. 动画编排系统

### 10.1 动画时序表（核心场景）

#### 场景 1：卡片展开到详情

| 时间轴 | 元素 | 动画 | 参数 |
|--------|------|------|------|
| 0.0s | 卡片 | 开始缩放 0.98 | `.spring(0.2)` |
| 0.1s | 背景遮罩 | 淡入（0 → 0.6） | `.easeIn(0.2)` |
| 0.15s | 卡片 | 生长到全屏 | `matchedGeometryEffect` |
| 0.3s | 背景渐变 | 从淡色到平台色 | `.easeInOut(0.4)` |
| 0.4s | 详情内容 | 淡入 + 上移 20pt | `.easeOut(0.3)` |
| 0.5s | 迷你曲线 | 展开成完整曲线 | `.spring(0.6)` |

#### 场景 2：下拉刷新

| 时间轴 | 元素 | 动画 | 参数 |
|--------|------|------|------|
| 拉动中 | 光晕 | 半径动态增长 | 实时跟手 |
| 触发 | 光晕 | 脉冲一次 | `.spring(0.3)` |
| 刷新中 | 渐变环 | 持续旋转 | `.linear.repeatForever` |
| 完成 | 粒子 | 爆散飘落 | 0.8s |
| 0.9s | 新内容 | 淡入 + 上浮 10pt | `.easeOut(0.4)` |

#### 场景 3：收藏操作

| 时间轴 | 元素 | 动画 | 参数 |
|--------|------|------|------|
| 0.0s | 心形 | 放大 1.0 → 1.3 | `.spring(0.2)` |
| 0.05s | 触觉 | 反馈 | `.impact(.medium)` |
| 0.1s | 颜色 | 空心 → 红色填充 | `.easeOut(0.2)` |
| 0.15s | 粒子 | 8-12个心形发射 | 初速度 300pt/s |
| 0.3s | 心形 | 回弹到 1.0 | `.spring(0.3)` |
| 1.0s | 粒子 | 完全消失 | `.easeIn(0.3)` |

### 10.2 动画曲线库

```swift
extension Animation {
    // 标准曲线
    static let quickResponse = Animation.spring(response: 0.3, dampingFraction: 0.7)
    static let smoothTransition = Animation.easeInOut(duration: 0.3)
    static let elasticPop = Animation.spring(response: 0.4, dampingFraction: 0.6)

    // 特殊效果
    static let gentleBreathing = Animation.easeInOut(duration: 3).repeatForever(autoreverses: true)
    static let pulsing = Animation.easeInOut(duration: 1.5).repeatForever(autoreverses: true)
}
```

## 11. 触觉反馈策略

### 11.1 触觉类型映射

| 操作类型 | 触觉类型 | 强度 | 使用场景 |
|----------|----------|------|----------|
| 轻量点击 | `.impact(.light)` | 弱 | Tab 切换、Toggle |
| 标准点击 | `.impact(.medium)` | 中 | 按钮、收藏 |
| 重要操作 | `.impact(.heavy)` | 强 | 长按、删除确认 |
| 成功反馈 | `.notification(.success)` | — | 刷新完成、保存成功 |
| 错误反馈 | `.notification(.error)` | — | 网络错误、操作失败 |
| 警告反馈 | `.notification(.warning)` | — | 屏蔽确认 |

### 11.2 触觉设计原则

- **不过度**：每屏最多 2-3 次触觉反馈
- **有意义**：只在关键状态变化时触发
- **可关闭**：设置中提供"触觉反馈"开关

// 防抖搜索
.onChange(of: searchText) { oldValue, newValue in
    searchTask?.cancel()
    searchTask = Task {
        try? await Task.sleep(for: .milliseconds(300))
        await performSearch(newValue)
    }
}

    LazyVStack {

### 14.2 减弱动态效果

```swift
@Environment(\.accessibilityReduceMotion) var reduceMotion

var animation: Animation? {
    reduceMotion ? nil : .spring(response: 0.3)
}
```

**降级策略：**
- 关闭：脉冲、呼吸、粒子动画
- 保留：基础过渡（但缩短时长）
- 替代：用透明度变化代替复杂动效
